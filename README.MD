1. Para traer los ultimos cambio:
git checkout main
git pull origin main

2. Crear tu propia rama:
git checkout -b NOMBRE_DE_TU_TAREA

3. Programar y Guardar (Local):
git add .
git commit -m "Mensaje corto de lo que agregaste o arreglaste"

4. Enviar tu tarea a GitHub:
git push origin NOMBRE_DE_TU_TAREA

Unir el trabajo al la rama principal:
Ir a github
darle al boton amarillo que dice "Compare & pull request"
Crear un pull request
Darle a merge
